FBLA-PROJECT  The Voyages of Christopher Columbus!

This project is a fun historical game about Christopher Columbus and his voyages. In the game take the role of Christopher Columbus and sail your ship to different areas of the world to explore them while collecting goods like gold. You find different tribes, some good and some bad. You have to complete tasks, satisfy the King and Queen of Spain by giving them some of your goods and also, survive.


Players Will:

-Explore islands

-Complete Tasks

-Interact with tribes

-Collect Resources

-Sail through the sea


Gameplay features:

-Multiple Voyages

-Complete Tasks

-Gather Goods

-Explore

-Interaction with NPCs


Built with Microsoft MakeCode Arcade


Controls: Arrow Keys/WASD, A button, B button, Menu button.


How to run the game: Open Microsoft MakeCode Arcade and wait for it to load.


Table of contents: Voyage 1, Voyage 2, Voyage 3


Usage: You can use the project to learn about Christopher Columbus and his voyages.
